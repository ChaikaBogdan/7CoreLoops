# 7CoreLoops
GitHub Pages for [7coreloops.net](https://7coreloops.net/)

# Development
1. Install Ruby 3.1+ [Windows installer](https://rubyinstaller.org) or [RVM for Linux](https://rvm.io/)
2. `gem install bundler`
4. `bundle install && bundle update --all`
5. `bundle exec jekyll serve --livereload`
6. Navigate to [http://localhost:4000](http://localhost:4000)
